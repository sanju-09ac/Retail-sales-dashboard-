# Sales Dashboard (Power BI + Excel)

##  Project Overview
This project is a Sales Dashboard created using Power BI and Excel to analyze and visualize sales performance.

The dashboard provides insights into:
- Total Sales
- Total Profit
- Sales by Region
- Sales Trend Over Time
- Category Performance
- Customer Analysis

---

##  Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Cleaning & Transformation
- Data Visualization
